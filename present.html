Module 2: Smart Analysis Engine

Raw data is useless without context. The Analyzer (`analyzer.py`) processes recon data to suggest attacks.

  • Input: Open ports, Technologies, WAF presence.
  • Logic:
    • If WAF Detected → Suggest Obfuscation tools.
    • If PHP + MySQL → Suggest SQL Injection scans.
  • Output: Confidence Score (0-100%).

def analyze_target(scan_data):
    confidence = 0
    if scan_data['waf_detected']:
        confidence -= 20 # Reduce aggression
        suggestions.append("Tamper Scripts")
    
    if "php" in scan_data['tech']:
        suggestions.append("SQLMap")
        
    return suggestions

Challenges & Roadmap

Technical Challenges

  • Concurrency: Preventing Celery workers from overlapping on the same target.
  • Parsing: Normalizing diverse outputs (JSON, XML, Text) from different tools.

Future Work

  • AI Integration: Use LLMs to interpret scan results naturally.
  • Reporting: Auto-generate PDF reports for compliance.

Conclusion

Intruder bridges the gap between manual pen-testing and automated compliance.
